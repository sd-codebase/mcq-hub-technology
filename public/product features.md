# QuizzyDock - User Features

**Master Programming Skills with 20,000+ Free Coding Questions**

## Core Features

1. **Three Test Formats**: Multiple Choice Questions (MCQ) with timed progression, Code Output Prediction to master exact output analysis, and Interview Scenario Questions for technical preparation.

2. **Comprehensive Subject Coverage**: Master JavaScript, TypeScript, React, Node.js, Angular, Vue.js, HTML, CSS, and Sass with structured topic-based learning paths.

3. **Organized Learning Structure**: Browse Subject → Topic → Subtopic hierarchy to find exactly what you need and track your learning progress systematically.

4. **Timed Testing System**: Timed questions with auto-progression, real-time feedback on correct/incorrect answers, and instant score calculations.

5. **Detailed Test Review**: Review all attempted questions with full explanations, visual answer indicators, and complete question context for effective learning.

6. **Auto-Test Playback**: Watch automated test demonstrations with smooth animations, intro/outro content, and question animations for passive learning.

7. **Daily Quiz Features**: Curated daily quizzes organized by topic, published test collections, and expandable sections for consistent learning habits.

8. **Free & No Sign-Up**: Completely free access to all quizzes and features with zero registration required to start learning immediately.

**Perfect for developers preparing for interviews, validating skills, and mastering programming concepts through practical coding challenges.**

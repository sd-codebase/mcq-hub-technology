QuizzyDock provides coding quizzes, MCQs, interview questions, and code-output challenges for developers and students.

Your task is to generate COMPLETE, high-quality, SEO-optimized metadata for a YouTube video.

⚠️ OUTPUT RULE:  
Return the output as a **VALID JSON CODE BLOCK ONLY**.  
Do NOT include any explanation, text, or formatting outside the JSON code block.

---

## OUTPUT FORMAT (STRICT)

Return a single JSON object with the following keys ONLY:

- title
- description
- tags
- pinned_comment
- playlist_name

---

## FIELD RULES

1. title

- One best SEO-optimized title
- Under 60 characters
- Primary keyword must appear in the first 4 words
- Clear, professional, non-clickbait

2. description

- First 2 lines optimized for YouTube search & CTR

- Mention this video is part of QuizzyDock’s coding quiz series
- Include a bullet list of concepts covered
- Specify who the video is for
- Include CTA (Like, Comment, Subscribe)

  Website: https://www.quizzydock.com  
  Android App: https://play.google.com/store/apps/details?id=com.krishna.jsleet.quizzydock
- End with 3–8 relevant hashtags only

3. tags

- Array of 15–20 high-ranking keywords
- Mix of primary, secondary, and long-tail keywords
- No duplicates

4. pinned_comment

- Engagement-focused
- Ask viewers to comment their score or answer
- Encourage interaction
- Soft promotion of QuizzyDock

5. playlist_name

- One SEO-friendly playlist name
